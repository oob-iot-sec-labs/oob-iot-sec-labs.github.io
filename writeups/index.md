# 📝 实战记录 | Writeups

收录 IOT 安全 CTF 题解与真实漏洞复现记录。

---

## 📋 收录规范

每篇 Writeup 请包含以下内容：

1. **目标设备/题目** — 设备型号或 CTF 题目名称
2. **漏洞类型** — 如 RCE、命令注入、未授权访问等
3. **分析过程** — 固件提取、逆向分析、漏洞定位
4. **利用方式** — PoC 代码或操作步骤
5. **修复建议** — 针对厂商或用户的加固建议
