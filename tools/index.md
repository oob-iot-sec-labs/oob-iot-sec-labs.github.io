# 🔧 安全工具 | Security Tools

收录 IOT 安全研究常用工具，涵盖固件分析、网络扫描、漏洞利用与防御加固。

---

## 📂 工具分类

### [固件分析](firmware-analysis/)
固件提取、解包、文件系统分析、静态/动态分析工具。  
*如：binwalk、Firmwalker、Ghidra、QEMU*

### [网络扫描](network-scanning/)
IOT 设备发现、端口扫描、协议分析工具。  
*如：nmap、masscan、Shodan CLI、ZMap*

### [漏洞利用](exploitation/)
针对 IOT 设备的 PoC、Exploit 框架及调试工具。  
*如：Metasploit 模块、RouterSploit、GDB*

### [防御加固](defense/)
IOT 设备安全加固、流量监控、威胁检测工具。  
*如：Snort、Suricata、IoTShield*
